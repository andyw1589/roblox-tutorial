Chapter 1: Printing

This chapter will introduce the basics in printing things to the output console.
To start, create a new game in Roblox Studio and open the command bar.

The most basic type of program you can make is to "print" something to the screen - in other words, make text appear.
In the command line, type in print("Hello, World!") and hit enter:

Ta-da! "Hello World" was just shown in the output console. Notice that we put our sentence in double quotes.

You can even print out math expressions:

You can see here that + is addition, - is subtraction, * is multiplication, and / is division. You can use brackets to specify the order of operations:

You can see that when printing (5 + 2) * 3, 5 + 2 is evaluated first, which is 7. Then it is multiplied, giving us 3.
On the other hand, when printing 5 + 2 * 3>, 2 * 3 is evaluated first because the order wasn't specified. So the result is 5 + 6 = 11. Without brackets, multiplication and division are evaluated before addition and subtraction.

Quick Exercise: What is the difference between (4 + 3) * (4 / 5) and 4 + 3 * 4 / 5? Print out the result.
